# 🔐 Password Manager

A simple and user-friendly Password Manager built using HTML, CSS, and JavaScript.

This application allows users to securely store and manage their credentials while providing an easy-to-use interface for password management.

---

## 🚀 Features

- Add website credentials
- Store usernames and passwords
- View saved passwords
- Copy passwords with one click
- Simple and responsive UI
- Built using pure HTML, CSS, and JavaScript
- Lightweight and easy to use

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Uzma1206/Password-Manager.git
```

2. Navigate to the project folder

```bash
cd Password-Manager
```

3. Open `index.html` in your browser.

---

## 💡 How It Works

1. Enter website details.
2. Enter username/email.
3. Enter password.
4. Save the credentials.
5. Access and copy saved passwords whenever needed.
